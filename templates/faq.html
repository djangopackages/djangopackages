{% translate "What are grids?" %}

{% blocktranslate %} Grids let you compare {{ FRAMEWORK_TITLE }} packages to each other. A grid comes with a number of default items compared, but you can add more features in order to get a more specific comparison. {% endblocktranslate %}

{% blocktranslate %} For now, we are trying out {{ SITE_TITLE }} without the traditional tagging system, because we think that grids give us a lot more specificity. {% endblocktranslate %}

{% translate "What are categories?" %}

{% blocktranslate %} Categories are broad groups of {{ FRAMEWORK_TITLE }} packages into something more granular than Grids. A package can only have one category. Changing then requires backend work, and is not done lightly. This is intentional, done in order to make searching easier. {% endblocktranslate %}
