Frequently Asked Questions

Who maintains this project?

Django Packages is maintained by Jeff Triplett and contributors just like you. Development is sponsored by REVSYS.

From 2010 to 2021, the majority of hosting and development was provided by Two Scoops Press, an imprint of Roy Greenfeld.

What role does the DSF have in this project?

Projects listed on Django Packages are third-party packages. They are not vetted nor endorsed by the Django Software Foundation. Use them at your own risk.

What are grids?

Grids let you compare {{ FRAMEWORK_TITLE }} packages to each other. A grid comes with a number of default items compared, but you can add more features in order to get a more specific comparison.

For now, we are trying out {{ SITE_TITLE }} without the traditional tagging system, because we think that grids give us a lot more specificity.

What are categories?

Categories are broad groups of {{ FRAMEWORK_TITLE }} packages into something more granular than Grids. A package can only have one category. Changing then requires backend work, and is not done lightly. This is intentional, done in order to make searching easier.

Which source control sites do you support?

GitHub, Bitbucket, and GitLab are supported.

What about RSS & Atom Syndication?

The Syndication page has more details.

How can I contribute?

This is an open source project, and we appreciate all sorts of contributions.

Where is Django Package hosted?

Django Package is hosted on DigitalOcean. If you would like to support us, please use our referral link and receive a free $200 credit to get you started.
